# MacOS setup

This repository contains my personal MacOS setup with all tools and configurations needed for my typical workflows.

## Setup guide

This guide goes through all required setup steps to complete the full MacOS setup.

### Run the `./setup.sh <private|work>` bash script

The `./setup.sh` bash script automatically sets up your MacOS.

**Set up private MacOS**
``` bash
./setup.sh private
```

**Set up work MacOS**
``` bash
./setup.sh work
```
